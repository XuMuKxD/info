git clone  https://... nameProject (клонирование проекта в папку)

Add files in commit:
git add .

Запись commit-а:
git commit -m "Comment commit"

Enter commit:
git push

Accept change:
git pull

Авторство commit-ов:
git config --global user.email "Your e-mail"
git config --global user.name "Your name"
